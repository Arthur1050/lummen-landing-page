## Implicações para a Landing

- A hero precisa abrir com a marca como protagonista e manter o slogan oficial.
- A busca deve ser um bloco funcional que direciona para o sistema atual de imóveis.
- Os diferenciais devem combinar emoção com segurança documental.
- O contato jurídico é um diferencial real e deve aparecer sem exagero.
- A equipe deve ser mostrada com nomes e links reais.
- Imagens e imóveis não devem ser substituídos por fotos genéricas quando os assets reais estiverem disponíveis.
